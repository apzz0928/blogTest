    고객을 360도로 이해하고 더 나은 고객 경험 구축을 위한
    데이터 통합 관리 솔루션 Dighty

    다이티(Dighty)는 기업이 가지고 있는 양질의 데이터와 비식별 형태의 제3자 데이터를 결합하여
    고객을 360도로 관찰하고, 정교한 타겟 마케팅까지 연결될 수 있도록 데이터 통합 관리 솔루션을 제공합니다.

  • 고객 360° View 비식별 형태의 1st Party Data와
    3rd party Data를 결합한
    고객 행동 데이터의 폭넓은 이해
  • 상용툴과의 손쉬운 연동 데이터 분석, 타겟 마케팅 목적에 따른
    손쉬운 데이터 공급과
    상용툴과의 연동 지원
  • 차별화된 솔루션 라인업 국내 대기업과의 협업으로 검증된
    NHN PAYCO / NHN ACE의
    데이터 통합 관리 솔루션

Dighty
Data Preparation & Marketing Management Platform

Data Manager
Data Preparation

다이티 데이터 매니저는 다양한 형태로 흩어져 있는 로그 및
데이터를 자동으로 전처리하는 기능을 제공하며 모든 데이터 처리 과정을
손쉽게 생성, 스케줄링하고, 모니터링할 수 있습니다.

데이터를 종합적으로 통제하고 관리할 수 있어
데이터 통합 관리(Data Governance)가 가능해집니다.

  • Data Governance Control
  • Data Preparation
  • Preprocessing Automation
  • Data Flow Managing

Audience Manager
Target Segmentation

다이티 오디언스 매니저는 고객의 특성 파악을 위한 오디언스를 분석합니다.

직접 타겟 조건을 선택하여 대상 오디언스를 추출하고,
추출된 오디언스 세그먼트는 고객 맞춤 캠페인을 위한 마케팅 툴
또는 캠페인 매니저에서 활용 가능합니다.

  • 오디언스 추출에 사용할 다양한 데이터 소스를 자유자재로 선택
  • 선택한 데이터를 기반으로 타겟 오디언스 모수 추출
  • 타겟 마케팅을 위한 세그먼트 별 분석 및 활용

Campaign Manager
Marketing Action

다이티 캠페인 매니저를 통해 고객의 특성과 관심사에 맞는
다양한 마케팅 메시지를 전달하세요.

오디언스 매니저에서 추출한 타겟에게 세그먼트 그룹별 고객 성향에 맞춘
캠페인을 세팅하고, 원하는 요일과 시간을 선택하여 운영할 수 있습니다.

  • 내 매체 안에서 캠페인 배너 영역 관리
  • 원하는 타겟에게 스케줄링을 통한 인하우스 캠페인 진행
  • 캠페인 별 노출 효과를 리포트로 확인

Dighty ASP

에이스카운터를 이용중인 고객님은 베타 테스트 기간동안 Dighty ASP 를 무료로 이용하실 수 있습니다.

오디언스 매니저에서 조건을 선택하여 오디언스를 추출하고,
캠페인 매니저에서 오디언스의 성향에 맞춘 인하우스 캠페인을 운영해보세요.

Data Providing by API Service

다양한 고객 행태정보를 API를 통해 제공받을 수 있습니다.

Data Driven Target Marketing

활용 사례

활용사례 01

"개인정보에 대한 리스크 없이 자사 회원정보와 연계한
의미있는 고객 데이터를 제공받고 싶습니다."

비식별 형태의 1st Party Data와 NHN ACE가 보유한
다양한 고객 행태 정보를 연계할 수 있습니다.
기업의 흩어진 데이터를 통합하여 비용과 기간을 줄이고
원하는 형태로 고객 데이터를 제공 받으세요.

활용 솔루션

  • Data Manager
  • Data API - Behavior

활용사례 02

"운영하는 서비스, 또는 매체에 원하는 타겟별로
배너광고를 송출하고자 합니다."

다이티 데이터 매니저를 통해 모델링된 데이터
또는 에이스카운터와의 연동을 통해 원하는 타겟 오디언스를 추출하세요.
기업이 보유한 매체를 통해 고객 맞춤형 캠페인을 운영하고자 할 경우 활용 가능합니다.

활용 솔루션

  • Dighty ASP, Data API-Segment
  • (Customizing, On-premise 구축 시)

    Data Manager, Audience Manager, Campaign Manager
